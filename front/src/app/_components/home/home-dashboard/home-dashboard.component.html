Welcome on SNIPPET

Snippet is intended for developers who want to keep small pieces of code aside in order to reuse them later.
Snippet version: 1.0

Features list:

- As guest and user, you can read existing snippets
- As user, you can create a new snippet
- As user, you can update your snippets
- As user, you can delete your snippets


As user, you can add a new file to your snippet
As user, you can delete a file from your snippet


- As user, you can add a new comment to existing snippets
- As user, you can delete your comments
- As user, you can update your comments


- As user, you can like a snippet
